# Elixir
**Create a Jekyll project folder with some extra ingredients.**

| Version |
|---------|
| 0.1.0  |

<br>

#### What's created:
- Jekyll directory
- Initialized as a Git repository (optional)
- Critical Inline CSS ready 
- manifest.json and browserconfig.xml
- Gemfile (optional)
- package.json (Gulp 4, Babel, and Browser-Sync ready)
- gulpfile.js
- LICENSE.md, README.md, and CONTRIBUTING.md
- robots.txt and sitemap.xml
- rss.xml

<br>

### Install

Yarn:

```shell
[sudo] yarn global add elixir
```

NPM:

```shell
[sudo] npm i -g elixir
```

#### Initiate Folder

```shell
mix dir <project-name>
```

#### Initiate Folder without Git

```shell
mix dir <project-name> --no-git
```

#### Initiate Folder without a Gemfile

```shell
mix dir <project-name> --no-gemfile
```
